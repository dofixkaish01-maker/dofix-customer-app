# Release Notes

## 16 July 2025

### New Features
- Guest Login: Users can now access the app as a guest without registration. Guest users have limited access and are prompted to register for full features (cart, account, etc.).

### Improvements
- Registration gating for App Store/Play Store compliance
- Permission dialogs and privacy strings updated
- UI fixes for login screen overflow and logout button reactivity

### Bug Fixes
- Minor UI and logic fixes for guest mode and registration flow
